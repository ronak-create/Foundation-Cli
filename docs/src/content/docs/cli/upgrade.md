---
title: foundation upgrade
description: Upgrade module versions in project.lock
---



Upgrade modules to current registry versions.

## Syntax

```bash
foundation upgrade [--dry-run]
```

## Options

| Option | Description |
|--------|-------------|
| `--dry-run` | Show what would be upgraded without applying |

## Note

Does NOT re-scaffold files — only updates lockfile.
