---
title: foundation eject
description: Copy module files into project for customization
---



Copy module files into your project so Foundation CLI stops managing them.

## Syntax

```bash
foundation eject [module-id]
```

## Example

```bash
foundation eject ui-tailwind
# Copies Tailwind files to project, removes from Foundation management
```

## Options

| Option | Description |
|--------|-------------|
| `--all` | Eject all modules |
