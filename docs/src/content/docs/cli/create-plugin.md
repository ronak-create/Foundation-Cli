Scaffold a new Foundation plugin package.

## Syntax

```bash
foundation create-plugin [name]
```

## Example

```bash
foundation create-plugin my-feature
```

## Output

```
foundation-plugin-my-feature/
├── manifest.json
├── hooks.mjs
├── package.json
├── tsconfig.json
├── src/index.ts
├── README.md
├── files/
└── patches/
```
